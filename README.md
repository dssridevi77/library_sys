# 📚 Digital Library Management System

## 📌 Project Description

This project is a Digital Library System developed using Node.js, Express, and MongoDB. It allows users to manage books with CRUD operations.

## 🚀 Features

- Add new books
- View all books
- Update book details
- Delete books

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Postman (for testing)

## ▶️ How to Run

1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open Postman:
   http://localhost:3000/books

## 📂 API Endpoints

- POST /addBook
- GET /books
- PUT /updateBook/:id
- DELETE /deleteBook/:id
